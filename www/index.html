Installation: Please read through the entire section before following the instructions.

Python 3.6
Download Anaconda for Python 3.6 which already includes most the popular scientific and numerical packages. Check that everything is installed by running python --version, conda update conda and jupyter notebook in the terminal\command line.
For the Cython and ctypes sessions (March 27, April 3)
Make sure that you have a C compiler installed:
  • Windows with Python version lower than 3.5: refer to these and these instructions.
  • Windows with Python 3.5/3.6:
    • Install MS Visual Studio 2015/2017 (the community version in good enough). You can do a custom install and select only C++ Visual Studio and Python Tools.
    • Add C:\Program Files (x86)\Microsoft Visual Studio 14.0\VC (or if it doesn't exist, whatever %VCINSTALLDIR% points to) to your PATH environment variable (or else, you should run C:\Program Files (x86)\Microsoft Visual Studio 14.0\VC\vcvarsall.bat in the command line console before starting the notebook server (jupyter notebook)).
  • Linux/Mac: Check if gcc is installed by running gcc --version, if gcc is not found, install it by:
    • On Ubuntu/Debian run sudo apt-get install build-essential;
    • On CentOS run sudo yum group install "Development Tools";
    • On Mac refer to these instructions.

Books:
  • Think Python by Allen B. Downey has basic, short, and clear explanations. Free ebook.
  • Dive Into Python by Mark Pilgrim is a free Python book for experienced programmers. Free ebook.
  • The IPython for Interactive Computing and Data Visualization by Cyrille Rossant targets those who already know Python, and want to learn IPython for the extended console, the notebook, and more advanced scientific applications.
  • Think Stats by Allen B. Downey is an introduction to Probability and Statistics for Python programmers. Free ebook.
  • Think DSP by Allen B. Downey is an introduction to Digital Signal Processing in Python.
  • Fluent Python by Luciano Ramalho reads fluently and makes you fluent in Python's features and details.
  • Flask Web Development by Miguel Grinberg is a terrific book on developing web apps and APIs with Flask.
